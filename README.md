# Hovoy GRE Pipe Web

Independent project for rebuilding the `hovoy-grepipe.com` business line as a standalone export-facing website.

## Current scope

- Dedicated brand site, separate from `lithostek-web`
- Initial homepage and message architecture
- Foundation for future product pages, technical data, and inquiry flow

## Recommended next steps

1. Create a dedicated GitHub repository: `hovoy-grepipe-web`
2. Add real product imagery and confirmed GRE product specifications
3. Build second-stage pages:
   - GRE Pipe Systems
   - Fittings and Fabrication
   - Joint Systems
   - Applications
   - About
   - Contact
4. Connect deployment to `hovoy-grepipe.com`
5. Upgrade from free email routing to hosted mailbox once inquiries start converting
